# enchf.github.io

Ernesto Espinosa — Personal portfolio website.

This repository contains the source for my personal website. The site will
feature the following main sections:

- About: short bio, education (EE 2026), and interests.
- Experience: work history, internships, and key accomplishments.
- Talks: list of speaking engagements, slides, and video links.
- Demos: interactive demos and runnable examples with instructions.
- Projects: portfolio projects with descriptions, tech stack, and links.

Development notes:

- Recommended stack: Next.js + Tailwind CSS + TypeScript.
- Local development (typical):

```bash
npm install
npm run dev
```
